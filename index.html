Product

Intuitive

Making mathematical formulas has never been so easy. Whereas typing difficult equations can quickly turn unclear, our drag'n'drop interface makes doing math bright as day.

Symbolic solutions

Everyday calculations are now smoother to execute, because Math DragOn supports both exact and approximate answers. This means that if you want π as an answer, you'll actually get π as an answer!

Primitives and derivatives

Not only Math DragOn be used for every day calculations, it can also be used to solve all derivatives, and most integrals, symbolically. Due to our drag'n'drop interface, they are easier to compose, too!

Open source

Math DragOn is open source and freely available under the LGPLv3 license.

Features

Build complex expressions

With secondary school maths there is no formula too strange or too big for the app, so it will be able to solve exercises you need help with.

Drag and drop interface

Build complex expressions using our easy to use drag and drop interface.

Easy keyboard input

Expressions need content. And with our intuitive keyboard you can easily insert variables and constants

Open and save formulas

If there is a formula of which you know you are going to need it a lot, you can bookmark it for later use.
