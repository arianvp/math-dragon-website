Math DragOn

The tool to help you with your math homework

Download now

Product

More understandable

Because you are putting the functions together yourself, you will get a better understanding of how the function works furthermore it will be a lot easier to maintain of overview of the function because it is not just one big line of numbers.

User friendly operating

With the use of a drag and drop interface and limited buttons the app is able to do all the functionalities used in secondary school but without the massive amount of options. You will never have to go through a dozen options to get the needed operator.

Open & save functions

The functionality to open functions used previously and save functions for later si a major component this app. The ability to do this will save you a lot of time, especially if you are studying a certain chapter with a limited amount of functions used again and again.

Features

Create your own functions

With secondary school maths there is no function too strange or too big for the app, so it will be able to help you to solve exercise you need help with.

Drag and drop interface for the operators

You will be able to make your own funtion with the slightest of ease.

Calculate a function exactly or ...

It does not matter whether the exercise is to calculate a function exactly or .. it, the app is capable of doing both. It can solve any problem the way you want it to be solved.

Open & save functions

If there is a function of which you know you are going to need a lot, you can make and save the funtion to open.
